YouTubeの子供向けチャンネルから通知がない理由

Jun 23, 2020

2020年の1月以降、YouTubeの子供向けチャンネルでは機能が制限された。児童オンラインプライバシー保護法(COPPA)に従った措置だ。しかし、この機能制限により不便になった。

通知が利用できなくなり、登録チャンネルから動画の更新を確認しなければならない。

COPPAが13歳未満の子供からデータの収集を禁止し、これに大人が巻き込まれたのだ。YouTubeは制限有りと制限なしのチャンネルが混在するようになった。
